# ANOIR EXPRESS — Téléchargements

Dépôt public de **distribution** de l'application mobile ANOIR EXPRESS.

> ⚠️ Ce dépôt ne contient **aucun code source**. Il sert uniquement à héberger les binaires de l'application (APK Android, IPA iOS), publiés automatiquement via GitHub Actions à partir du dépôt privé.

## Télécharger l'application

Rendez-vous dans l'onglet **[Releases](../../releases)** pour récupérer la dernière version :

- **Android** : fichier `.apk` — téléchargez-le et installez-le directement sur votre appareil (autorisez les sources inconnues).
- **iOS** : fichier `.ipa` — build non signé (installation via un canal signé : TestFlight ou MDM).

Chaque release indique la version et le numéro de build.
